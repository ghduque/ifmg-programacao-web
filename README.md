Repositório destinado à disciplina de **Programação Web**, do Instituto Federal de Minas Gerais (IFMG).

Este projeto reúne exercícios de aula, listas de tarefas e trabalhos práticos desenvolvidos ao longo do semestre, abordando a construção de interfaces interativas e o desenvolvimento Backend dinâmico com **PHP**.

##  Tópicos Abordados

Abaixo, a organização dos conteúdos estudados e implementados neste repositório:

### Frontend (Interface)
-  HTML5 Semântico
-  CSS3 (Layouts, Flexbox, Grid)
-  JavaScript (DOM, Eventos)
-  Design Responsivo

### Backend (Servidor)
-  Sintaxe e Lógica com PHP
-  Processamento de Formulários (GET/POST)
-  Gerenciamento de Sessões e Cookies
-  Conexão com Banco de Dados (PDO/MySQL)
-  Arquitetura MVC (Básico)

### Ferramentas e Práticas
-  Versionamento com Git
-  Ambiente de Desenvolvimento (XAMPP/Laragon)
-  Validação de Dados

##  Tecnologias Utilizadas

Os projetos foram desenvolvidos utilizando a stack fundamental da web:

* **[HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)**: Estruturação de conteúdo.
* **[CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)**: Estilização e layout.
* **[JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)**: Interatividade no navegador.
* **[PHP](https://www.php.net/)**: Processamento server-side e regras de negócio.
